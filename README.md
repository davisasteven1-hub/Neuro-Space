# NeuroSpace built with Google AI Studio

> It's basically a timer that counts down the time remaining until an exam.

It doesn't have any AI feature as of v1.0

## Run Locally

This contains everything you need to run your app locally.

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
